# Bagus Abadi Las (BAL)

Website profil perusahaan (Company Profile) untuk **Bagus Abadi Las**, sebuah bengkel las profesional yang berlokasi di Citeureup, Bogor. Website ini dirancang untuk menampilkan layanan, galeri pekerjaan, dan informasi kontak agar mudah diakses oleh calon pelanggan.

## Fitur Utama

*   **Beranda Menarik**: Hero section dengan deskripsi singkat dan tombol ajakan bertindak (CTA).
*   **Daftar Layanan**: Informasi mengenai layanan pembuatan Kanopi, Pagar, Pintu Besi, Teralis, dan Railing.
*   **Galeri Pekerjaan**: Menampilkan foto-foto hasil pengerjaan yang terintegrasi dengan sumber gambar Google Maps.
*   **Kontak & Lokasi**: Tautan langsung ke WhatsApp untuk pemesanan dan Google Maps untuk navigasi lokasi.
*   **Desain Responsif**: Tampilan yang optimal di perangkat desktop maupun mobile (menggunakan Tailwind CSS).

## Teknologi yang Digunakan

*   HTML5
*   Tailwind CSS (via CDN)
*   Font Awesome (untuk ikon)
*   Google Fonts (Poppins)

## Cara Penggunaan

Cukup buka file `index.html` di browser web modern apa pun. Tidak diperlukan instalasi atau build step khusus karena menggunakan CDN untuk styling.
